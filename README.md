# Simple PHP GPA Calculator
<img width="1920" height="1080" alt="Gpa Calc Image" src="assets/ss.png" />
A simple gpa calc built as a university course assignment to calculate a student's GPA based on 3 courses.

## What it does
* Input grades for 3 distinct courses.
* Validates inputs to ensure accurate calculations.
* Displays final GPA instantly.

## Tech Stack
* PHP (Vanilla)
* HTML5 / CSS3
